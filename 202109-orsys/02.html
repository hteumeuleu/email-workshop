Découvrir les trois approches pour une stratégie email mobile

  1. Scalable
  2. Responsive
  3. Fluide/hybride
  4. Mobile first

Scalable

ou « Youpi ! Il n'y a rien à faire ! »

Scalable

On optimise le design pour un affichage redimensionné. On intégre l'email en version desktop. Et on laisse le client mail mobile adapter le rendu.

  • Exemples de MeUndies, Zara

Scalable

On utilise <meta name="x-apple-disable-message-reformatting"> pour Apple Mail sur iOS. (exemples)

Scalable

C'est plus du design que du code. Il faut faire les choses en grand.

Néanmoins, on reste à la merci du comportement des clients mail.

  • Human Interface Guidelines, Apple

Responsive

ou « On commence en desktop et on utilise des media queries pour le mobile. »

Media query

@media only screen and (max-width:600px) {
	.un-titre-dans-mon-email {
		color: red;
	}
}

Deux classes utilitaires pratiques

  1. .mobile-block { display:block; }
  2. .mobile-width { width:100%; }
